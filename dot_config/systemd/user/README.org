#+TITLE: How to setup systemd user service
#+AUTHOR: Rachit Kumar Verma

=NOTE=: To set up swayidle for Niri or any other compositor where uwsm is not used, use swayidle2.service

* Setup swayidle service:
#+begin_src bash
  $mkdir -p $HOME/.config/systemd/user # First create the systemd directory 
  $cd $HOME/.config/systemd/user # Then cd into the directory 
  $aria2c https://raw.githubusercontent.com/rachitve6h2g/dotfiles/refs/heads/main/dot_config/systemd/user/swayidle2.service # Use the raw link to download the file

  # Then activate the user service:
  $systemd --user daemon-reload
  $systemd --user enable swayidle2.service
#+end_src
